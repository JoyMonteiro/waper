# Phase 3 Algorithmic Improvements Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Refactor the identification algorithms in the WAPER package to replace Affinity Propagation with DBSCAN, fix geometric scaling bugs, improve path ranking, and refine handling of periodic longitude boundaries.

**Architecture:** We will systematically modify `topology.py`, `rwp_graph.py`, and `utils.py`, backing up the algorithmic changes with pytest cases. We will consolidate redundant max/min functions into unified generic functions where possible.

**Tech Stack:** Python 3.9+, numpy, scipy, scikit-learn, networkx.
